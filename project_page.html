GestureCommand: A Camera-Based Gesture Recognition System for Autonomous Table-Specific Delivery Robot

Chit Lee*, Juni Katsu*, Cheuk Yu Lam*, Abbas Mandasorwala*, Kozerenko Elizaveta*
University of Birmingham
Intelligent Robotics Final Team Project
*Indicates Equal Contribution
Paper Code

Abstract

This report details the development of a robot's software, designed to improve service in cafe environments through two key features: gesture control and navigation. The gesture control system enables staff to direct the robot to specific tables, and for customers - to direct the robot back to the till with simple hand gestures, providing an intuitive and interactive method for beverage delivery. Meanwhile, the navigation feature ensures an accurate and efficient path finding of the robot from the till to the designated table and back. This innovative combination aims to enhance operational efficiency, reduce service time, and improve the overall customer experience in cafes. The report covers the technical aspects of the software, including the design of the gesture recognition algorithms, the implementation of navigation techniques, and the challenges encountered during the development process. Additionally, it explores the possible improvements that can be made to the software, so it works better.
